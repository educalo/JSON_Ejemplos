JSON (JavaScript Object Notation)

Es un estándar basado en texto plano para el intercambio de información. La ventaja de JSON es que los servicios que comparten información por éste método, no necesitan hablar el mismo idioma, es decir, el emisor puede ser Java y el receptor PHP.

Para entender mas como interactuar la data JSON con la vista del navegador se recomienda revisar los siguientes archivos:

  • revisar el Readme del proyecto.
  • Revisar el archivo ./src/index.html
  • Revisar el archivo ./src/js.js
